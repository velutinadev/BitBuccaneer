# BitBuccaneer 🏴‍☠️

A clean, native Mac app for searching torrents — without the ads, pop-ups, and general chaos of torrent websites.

I built this because I was tired of torrent sites. You know the ones — aggressive advertising, sketchy pop-ups, and three clicks before you can even see a search result. BitBuccaneer cuts all of that out. Type what you're looking for, get results immediately.

## What it does

Searches YTS, Nyaa, LimeTorrents, and TorrentDownloads simultaneously and shows you everything in one clean list. Click a result to open it in your torrent client. That's it.

## Features

- Search multiple sources at once
- Filter by category — Movies, TV, Music, Games, Software, Anime
- Sort by seeders, leechers, size, or name
- Movie posters for YTS results
- Search history
- Copy magnet links directly
- Liquid Glass UI inspired by macOS design guidelines

## Installation

1. Download `BitBuccaneer-1.0.0.dmg` from the Releases page
2. Open the DMG and drag BitBuccaneer to your Applications folder
3. Right-click → Open on first launch (app is unsigned)

## Legal

BitBuccaneer searches public torrent indexers. It does not host any content. Always respect copyright laws in your jurisdiction.
